🪄 ServerStick

One-command self-hosting. Turn any Linux box into a server with a public HTTPS dashboard.

To install, run this in a terminal on the target machine:

curl -fsSL https://get.serverstick.com/install.sh | sudo bash

Then open http://<box-ip>:18090 in your browser to complete the wizard.

What it does:
  1. Installs Docker, Node 22, Newt (Pangolin tunnel client)
  2. Deploys hermes-bridge (FastAPI + Svelte dashboard) on port 18090
  3. Wizard picks your subdomain, AI tier, and connected services

For the static install script: /install.sh
